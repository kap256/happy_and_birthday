::
::    フルーツメソッド ( :fruit )
::    　　　ObjNo 10~30 の Obj に対してのみ適用可
::
::MFlag[8]---入手できるフルーツの下限
::MFlag[9]---入手できるフルーツの種類の数
::MFlag[10~30]フルーツを入手したかどうか
::MFlag[31]---使用
::
:::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::

:fruit
Store MFlag[31],ParamA;
If MFlag[MFlag[31]],0,!=,:fruit_end ;
GetItem Rand%MFlag[9]+MFlag[8] ;
Store MFlag[MFlag[31]],1;
:fruit_end
End;
